# Установка на MetaTrader 5 — пошаговое руководство (на русском)

Это руководство описывает установку советника (EA), индикатора и пресетов на
MetaTrader 5 с русским интерфейсом.

> ⚠️ **ВАЖНО:** это не «готовый заработок». Советник открывает и закрывает сделки
> автоматически, но **гарантии прибыли НЕТ**. Сначала — тестер и ДЕМО-счёт, только
> потом реальный счёт с малым риском.

---

## Что вы устанавливаете

| Файл | Тип | Куда |
|---|---|---|
| `SMC_AutoTrader.mq5` | Советник (Эксперт) | `MQL5/Experts/` |
| `BuySellSignal.mq5` | Индикатор | `MQL5/Indicators/` |
| `presets/*.set` (8 шт.) | Пресеты (настройки) | `MQL5/Presets/` |

---

## Шаг 1. Скачать файлы с GitHub

1. Откройте репозиторий: `github.com/Doniyorbek121/error`
2. Переключите ветку на `claude/shu-error-metatredir-indicator-2cvke5`
3. Зелёная кнопка **Code → Download ZIP**
4. Распакуйте архив на компьютере

Вам нужны: `SMC_AutoTrader.mq5`, `BuySellSignal.mq5` и папка `presets` с файлами `.set`.

---

## Шаг 2. Открыть папку данных MT5

1. Откройте MetaTrader 5
2. Меню: **Файл → Открыть каталог данных**
   (англ.: File → Open Data Folder)
3. В открывшемся окне зайдите в папку **`MQL5`**

Внутри увидите папки: `Experts`, `Indicators`, `Presets` и другие.

---

## Шаг 3. Скопировать файлы в нужные папки

Скопируйте (Копировать → Вставить):

- `SMC_AutoTrader.mq5` → в папку **`MQL5\Experts`**
- `BuySellSignal.mq5` → в папку **`MQL5\Indicators`**
- Все файлы `.set` из `presets` → в папку **`MQL5\Presets`**

> 💡 Если папки `Presets` нет — создайте её (правой кнопкой → Создать папку → `Presets`).

---

## Шаг 4. Компиляция (обязательно!)

1. В MT5 нажмите **F4** — откроется **MetaEditor** (Редактор)
2. Слева в окне **«Навигатор»**: `Experts` → двойной клик по `SMC_AutoTrader.mq5`
3. Нажмите кнопку **«Компилировать»** (или **F7**)
4. Внизу должно появиться: **«0 ошибок, 0 предупреждений»** (0 errors, 0 warnings) ✅
5. То же самое сделайте для `BuySellSignal.mq5`

> ⚠️ Если появятся **ошибки (errors)** — пришлите мне текст, я сразу исправлю.
> Без успешной компиляции советник работать не будет.

---

## Шаг 5. Обновить список в MT5

Вернитесь в MT5. В окне **«Навигатор»** (Ctrl+N):
- **Советники (Expert Advisors)** → появится `SMC_AutoTrader`
- **Индикаторы (Indicators)** → появится `BuySellSignal`

Если не видно — правой кнопкой в Навигаторе → **Обновить**.

---

## Шаг 6. Открыть график

1. Вверху выберите инструмент — например **XAUUSD** (или `GOLD` у вашего брокера)
2. Таймфрейм: **M15**
3. (По желанию) перетащите индикатор `BuySellSignal` на график

---

## Шаг 7. Запустить советник

1. Включите кнопку **«Алготрейдинг»** вверху (должна стать зелёной)
2. Перетащите `SMC_AutoTrader` из Навигатора **на график**
3. В открывшемся окне:
   - Вкладка **«Общие»** → поставьте галочку **«Разрешить алготрейдинг»**
   - Вкладка **«Входные параметры»** → внизу кнопка **«Загрузить»** → выберите
     `SMC_XAUUSD_M15.set`
4. Нажмите **OK**

Если в правом верхнем углу графика появился значок 🙂 — советник работает.
Слева появится панель с информацией (тренд, зона, сделки, прибыль).

---

## Шаг 8. Тест перед реальной торговлей (обязательно!)

### 8.1 Тестер стратегий
1. Меню **Вид → Тестер стратегий** (View → Strategy Tester)
2. Советник: `SMC_AutoTrader`, Символ: `XAUUSD`, Период: `M15`
3. Вкладка входных параметров → **«Загрузить»** → пресет `.set`
4. Задайте период (например, последние 1–2 года) → **Старт**
5. После теста смотрите вкладку **«Результаты / Бэктест»**

### 8.2 ДЕМО-счёт
После хорошего теста запустите на **ДЕМО-счёте** и наблюдайте несколько недель
в реальном времени.

---

## Правильный порядок действий

```
1. F7 (компиляция)              ← начните отсюда
2. Тестер стратегий             ← проверка на истории
3. ДЕМО-счёт (недели)           ← наблюдение в реальном времени
4. Реальный счёт (малый риск)   ← только если шаги 1–3 успешны
```

---

## Частые проблемы

| Проблема | Решение |
|---|---|
| Значок 🙂 не появляется | Включите кнопку «Алготрейдинг» + галочку «Разрешить алготрейдинг» |
| «Автоторговля запрещена» | Сервис → Настройки → Советники → разрешить |
| Советник не открывает сделки | Проверьте вкладку «Эксперты» внизу — там пишется причина (спред, лимит и т.д.) |
| Ошибка при компиляции | Пришлите текст ошибки — исправлю |
| Символ не `XAUUSD` | У брокера может быть `GOLD`, `XAUUSD.m` — выберите свой |

---

## ⚠️ Помните

- Гарантии прибыли нет. Качество кода ≠ прибыль.
- Сначала ДЕМО, потом реальный счёт с малым риском (1% или меньше).
- Каждый инструмент тестируйте отдельно.
